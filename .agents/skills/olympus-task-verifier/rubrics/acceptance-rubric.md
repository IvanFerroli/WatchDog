# Acceptance Rubric

## Objetivo
Avaliar se uma task executada satisfaz, de forma objetiva, os critérios de aceite e a Definition of Done.

## Critérios
Pontue cada item de 1 a 5.

1. Objetivo único cumprido
O resultado entregue cumpre a essência da task?

2. Acceptance Criteria atendidos
Os critérios de aceite foram satisfeitos de forma verificável?

3. Definition of Done atendida
A task atingiu o padrão de completude esperado?

4. Validação observável
A validação foi executada e demonstrada?

5. Aderência ao escopo
O resultado respeita alvos e fora de escopo?

6. Consistência do relatório
O relatório de execução bate com o que a task pedia?

7. Qualidade do retorno
O feedback de retorno é útil para o próximo ciclo?

## Interpretação sugerida
- 30 a 35: forte candidata a aprovado
- 24 a 29: pode cair em aprovado com ressalvas
- 18 a 23: tende a reprovado
- abaixo de 18: não aceitar

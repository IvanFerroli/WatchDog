# TypeDoc Readiness Rubric

## Objetivo
Avaliar se o projeto ou uma parte dele já está pronta para documentação viva baseada em TSDoc/TypeDoc.

## Critérios
Pontue de 1 a 5.

1. Boundaries de módulos definidos
2. Contratos públicos estabilizados
3. Convenção documental minimamente definida
4. Nível de churn aceitável
5. Clareza das superfícies públicas
6. Baixo risco de documentar ruído
7. Capacidade de manter documentação viva

## Interpretação
- 28 a 35: pronto para iniciar
- 21 a 27: quase pronto
- 14 a 20: cedo demais
- abaixo de 14: não iniciar

# Execution Readiness Rubric

## Objetivo
Avaliar se uma task está pronta para entrar em execução operacional.

## Critérios
Pontue de 1 a 5.

1. Clareza da task
2. Objetivo único real
3. Alvos explícitos
4. Dependências satisfeitas
5. Fora de escopo claro
6. Validação observável
7. Evidência esperada definida
8. Risco estrutural controlado
9. Compatibilidade com o canônico
10. Tamanho adequado para ciclo curto

## Classificação final
- 40 a 50: pronta para execução
- 30 a 39: pronta com ressalvas
- 20 a 29: fraca, voltar para ajuste
- abaixo de 20: não executar

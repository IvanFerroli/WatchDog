# Routing Rubric

## Objetivo
Escolher o modo de execução mais adequado para uma task já definida.

## Critérios
Pontue cada modo candidato de 1 a 5.

1. Aderência do modo ao objetivo da task
2. Clareza operacional do modo
3. Menor ambiguidade
4. Menor risco de cruzar superfícies demais
5. Facilidade de validação
6. Facilidade de evidência
7. Menor chance de abrir frente paralela
8. Compatibilidade com a atomicidade da task

## Resultado
- maior pontuação: modo preferencial
- empate: escolher o modo mais simples
- ambiguidade forte: devolver ao Taskyfier para quebra ou clarificação

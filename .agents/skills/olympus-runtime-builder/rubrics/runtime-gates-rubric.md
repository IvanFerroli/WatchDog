# Runtime Gates Rubric

## Objetivo
Avaliar se uma superfície do Olympus Climb já pode receber materialização runtime.

## Gates
Pontue cada gate de 1 a 5.

1. Gate de contrato
2. Gate de spec
3. Gate de task
4. Gate de qualidade mínima
5. Gate de vertical mínima

## Interpretação
- 22 a 25: pronto para materialização
- 17 a 21: quase pronto, revisar ressalvas
- 11 a 16: cedo demais
- abaixo de 11: não materializar
